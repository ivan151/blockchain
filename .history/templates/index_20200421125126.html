Simple blockchain system

Hi! This site is created to show how blockchain works. The main idea of blockchain is that every action of its members is written to the block. And every next block keeps information about previous block’s «hash». That’s why you can’t violate the integrity of the chain. Each block in such system is forever That’s what this site demonstrates

To understand this better, create a new user or add a transation to blockchain. You will see how every your action will create a new block with hash of the previous one.

Check integrity of the blocks.

Let's say you are hacker. Try to corrupt some block. Even if you're lucky and make it, the system will repair itself

But what if you corrupt the last block? That means you can then change each block one by one. Nope. It's not that simple because of Nodes. Every node in the system remembers this last block. To corrupt it you should to have an access to more than half of the nodes. Otherwise the system will just ignore modified block

Ok. Now, when you know it, try to corrupt the last block in more than half of the nodes. It's not a corruption anymore! This is a legal change with the consent of the majority of participants.
